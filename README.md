## Documentation Source of Truth

Markdown files in this repository are the durable project source of truth.

Supporting tools such as ChatGPT, Claude, Obsidian, VS Code, OneNote, Teams, and local LLMs may assist with drafting and reviewing, but Git-tracked Markdown files should remain the authoritative project record.
